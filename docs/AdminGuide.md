# Admin Guide

## First login

- Create the bootstrap admin with:
  - `BootstrapAdmin:Email`
  - `BootstrapAdmin:Password`

## Common tasks

- Manage users from `Users`
- Manage roles and permissions from `RolesManagement` and `Security`
- Review audit logs from `AuditLogs`
- Check system status from `/health`

## Security notes

- Use strong passwords in production.
- Enable 2FA at the Identity layer before go-live.
- Review the audit dashboard daily.
